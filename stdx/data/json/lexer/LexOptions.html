    Enum LexOptions

    Flags for configuring the JSON lexer.

    enum LexOptions : int { ... }

    These flags can be combined using a bitwise or operation.

    Enum members

    NameDescription
    init Default options - track token location and only use double to represent numbers
    noThrow Uses JSONToken.Kind.error instead of throwing exceptions
    noTrackLocation Counts lines and columns while lexing the source
    specialFloatLiterals Support "NaN", "Infinite" and "-Infinite" as valid number literals
    useBigInt Use BigInt to represent integers (if larger than long or useLong is not given)
    useLong Use long to represent integers
