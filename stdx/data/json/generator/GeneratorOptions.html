    Enum GeneratorOptions

    Flags for configuring the JSON generator.

    enum GeneratorOptions : int { ... }

    These flags can be combined using a bitwise or operation.

    Enum members

    NameDescription
    compact Avoid outputting whitespace to get a compact string representation
    escapeUnicode Output all non-ASCII characters as unicode escape sequences
    init Default value - enable none of the supported options
    specialFloatLiterals Output special float values as 'NaN' or 'Infinity' instead of 'null'
